# Utility App

A utility-focused Node.js application project.

## Setup

1. Install dependencies:
   `npm install`
2. Start/build as needed using scripts from `package.json`.

## Build

1. Install dependencies:
   `npm install`
2. Build the project:
   `npm run build`

## Push Changes

1. Check status:
   `git status`
2. Stage files:
   `git add .`
3. Commit:
   `git commit -m "Your commit message"`
4. Push to main:
   `git push origin main`

## Notes

- Environment variables are stored in `.env` (not committed).
- Build artifacts and dependencies are excluded via `.gitignore`.
